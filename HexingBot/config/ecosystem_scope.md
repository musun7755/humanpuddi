# Ecosystem Scope

每日只觀察適合赫湦轉化的輕量社群趨勢。

觀察類型：
- 流行語
- meme 形式
- 生活心情
- 可愛廢文
- 寵物梗
- 創作者日常
- 食物與生活小事
- 輕事件或大家正在討論的小現象

不做長篇分析。
不搬運原文。
不複製別人的句子。
只整理成「今天大家大概在聊什麼」和「赫湦可以怎麼轉化」。
